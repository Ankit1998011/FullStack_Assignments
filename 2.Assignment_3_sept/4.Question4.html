Ordered List of HTML tags:-

  1. html: The root element of an HTML document that defines the document type. All other tags are contained within this tag.
  2. head: The container for the metadata of the document, including title, CSS stylesheets, JavaScript, character encoding, and more. It is not visible on the webpage itself.
  3. title: Specifies the title of the webpage, which appears in the browser's title bar or tab.
  4. body: Contains the visible content of the webpage, including text, images, links, forms, etc.
  5. h1 to h6: Heading tags that define different levels of headings, with h1 being the highest level and h6 the lowest. They are used to structure the content and indicate the hierarchy of information.
  6. p: Represents a paragraph of text
  7. a: Creates a hyperlink or anchor, allowing users to navigate to another webpage, a different section within the same page, or an external resource.
